## myrc
> 这是一些通用的配置 可以通过软链接引用

### 拉取代码
```bash
$ git clone git@github.com:yinshiyionly/myrc.git .myrc
```

### 使用

1. 取代 `.zshrc`
```bash
$ ln -s ~/.myrc/oh-my-zsh/zshrc ~/.zshrc
```

2. 取代 `.gitconfig`
```bash
$ ln -s ~/.myrc/gitconfig/gitconfig ~/.gitconfig
```

3. 取代 `.ssh`
```bash
$ ln -s ~/.myrc/ssh ~/.ssh
```
